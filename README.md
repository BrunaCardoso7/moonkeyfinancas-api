## Description

Api RestFull desenvolvida com Nest.js e GraphQl, tem como proposta gestão de vendas e pedidos focado no ramo comercial. Com ela, posso registrar, atualizar, deletar e consultar pedidos e vendas.

## Tecnologias

 - <b>Nest js</b>
 - <b>TypeORM</b>
 - <b>Typescript</b>
 - <b>GraphQL</b>
 - <b>JWT</b>
